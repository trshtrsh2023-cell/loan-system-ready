## 1️⃣ حساب العمر والخدمة

### في Excel:
```
P4 = (E7-E5)*12+E6-E4     // العمر بالأشهر
P6 = (E7-E9)*12+...       // مدة الخدمة
```

### في Code:
```typescript
const currentYear = new Date().getFullYear()
const currentAge = currentYear - birthYear
const currentAgeMonths = currentAge * 12 + (currentMonth - birthMonth)
const serviceMonths = (appointmentYear - currentYear) * 12 + ...
```

**الفائدة:** إحنا نحسب الأشهر بدقة عشان نعرف كم باقي للتقاعد.

## 2️⃣ سن التقاعد (المهم جداً!)

### المدني (حكومي + خاص):
```
سن التقاعد = 60 سنة = 720 شهر
```

### العسكري (حسب الرتبة):
| الرتبة | السن | الأشهر |
|-------|------|--------|
| جندي | 44 | 528 |
| عريف | 49 | 588 |
| وكيل رقيب | 50 | 600 |
| رقيب / رقيب أول / رئيس رقباء | 52 | 624 |
| ملازم | 48 | 576 |
| نقيب | 53 | 636 |
| رائد | 55 | 660 |

### في Code:
```typescript
export const MILITARY_RANKS = [
  { value: 'جندي', retirementAge: 44 },
  { value: 'عريف', retirementAge: 49 },
  // ...
]

function getRetirementAge(jobType: string, militaryRank?: string): number {
  if (jobType === 'military') {
    const rank = MILITARY_RANKS.find(r => r.value === militaryRank)
    return rank?.retirementAge ?? 52
  }
  return 60 // مدني
}
```

## 3️⃣ المتبقي للتقاعد (الحساب الذكي!)

### المنطق:
```
أشهر_المتبقي = (سن_التقاعد × 12) - العمر_بالأشهر

مثال:
- عسكري رتبة جندي: سن_التقاعد = 44 سنة = 528 شهر
- عمره = 30 سنة = 360 شهر
- المتبقي = 528 - 360 = 168 شهر (14 سنة)
```

### في Code:
```typescript
const retirementAge = getRetirementAge(jobType, militaryRank)
const yearsToRetirement = retirementAge - currentAge
const monthsToRetirement = yearsToRetirement * 12
```

## 4️⃣ معامل الضرب الذكي ⭐

### هذا البيت الأساسي!

**القاعدة:**
- إذا باقي <= 5 سنوات (60 شهر): **اضرب معامل بنسبة المتبقي**
- إذا باقي > 5 سنوات: **اضرب معامل كامل**

### مثال:
```
شخص عمره 58 سنة (ستيل 2 سنة للتقاعد)
- معامل البنك = 18
- المتبقي = 24 شهر
- معامل_فعلي = 18 × (24 ÷ 60) = 18 × 0.4 = 7.2
- قرضه = راتبه × 7.2 (وليس × 18!)
```

### في Code:
```typescript
const maxPeriod = Math.min(monthsToRetirement, cfg.max_period_months)

const effectiveMultiplier = maxPeriod >= cfg.max_period_months
  ? cfg.personal_multiplier  // 18 كامل
  : cfg.personal_multiplier * (maxPeriod / cfg.max_period_months)  // تناسبي
```

## 5️⃣ مبلغ التمويل الأساسي

### الصيغة البسيطة:
```
مبلغ_التمويل = صافي_الراتب × معامل_الضرب_الفعلي
```

### مثال:
```
- صافي الراتب = 10,000 ر.س (أساسي 8000 + سكن 2000)
- معامل = 18
- مبلغ التمويل = 10,000 × 18 = 180,000 ر.س
```

### في Code:
```typescript
const netSalary = basicSalary + housingAllowance + otherAllowances
const baseLoan = Math.floor(netSalary * effectiveMultiplier)
```

## 6️⃣ دعم البرنامج السكني

### القاعدة:
```
إذا الراتب < 10,000:    دعم = 150,000 ر.س
إذا الراتب >= 10,000:   دعم = 100,000 ر.س
```

### في Code:
```typescript
let sakaniAmount = 0
if (input.sakaniSupport) {
  sakaniAmount = input.netSalary < cfg.sakani_low_threshold
    ? cfg.sakani_low_support        // 150,000
    : cfg.sakani_high_support       // 100,000
}
```

## 7️⃣ القسط الشهري (PMT Formula)

### الصيغة الرياضية (من Excel):
```
القسط = المبلغ × (نسبة × (1+نسبة)^عدد_الأشهر) / ((1+نسبة)^عدد_الأشهر - 1)

مثال:
- مبلغ = 180,000 ر.س
- نسبة شهرية = 2.5% ÷ 12 = 0.002083
- مدة = 60 شهر
- القسط = 180,000 × (0.002083 × 1.002083^60) / (1.002083^60 - 1)
        ≈ 3,100 ر.س شهرياً
```

### في Code:
```typescript
const monthlyRate = cfg.annual_rate / 100 / 12
const factor = Math.pow(1 + monthlyRate, maxPeriod)
const installment = Math.ceil(
  totalLoan * monthlyRate * factor / (factor - 1)
)
```

## 8️⃣ التحقق من الاستطاعة (Safety Check)

### القاعدة:
```
القسط_الشهري يجب ألا يتجاوز: راتب × نسبة_الاستقطاع

مثال:
- راتب = 10,000 ر.س
- نسبة_استقطاع = 33%
- أقصى_قسط = 10,000 × 0.33 = 3,300 ر.س
- إذا القسط > 3,300: تحذير!
```

### في Code:
```typescript
const maxAllowedInstallment = Math.floor(netSalary * cfg.deduction_rate)
const note = installment > maxAllowedInstallment
  ? `القسط الشهري ${installment} ر.س يتجاوز الحد المسموح`
  : undefined
```

## 9️⃣ مثال كامل (حساب حقيقي)

### البيانات المدخلة:
```
الاسم: أحمد محمد
سنة الميلاد: 1990 (العمر: ~34)
سنة التعيين: 2015 (خدمة: ~9 سنوات)
الوظيفة: عسكري، رتبة: جندي
الراتب الأساسي: 5,000
بدل السكن: 1,500
بدلات أخرى: 500
صافي الراتب: 7,000
دعم سكني: نعم
البنك: الأهلي (معامل 18)
```

### الحسابات:

**الخطوة 1: العمر والمتبقي**
```
العمر = 2024 - 1990 = 34 سنة
الخدمة = 2024 - 2015 = 9 سنة
سن_التقاعد (جندي) = 44 سنة
المتبقي = 44 - 34 = 10 سنوات = 120 شهر
```

**الخطوة 2: معامل الضرب**
```
المتبقي (120) > 60، لذا: معامل_فعلي = 18 كامل
```

**الخطوة 3: مبلغ التمويل**
```
مبلغ_أساسي = 7,000 × 18 = 126,000 ر.س
دعم_سكني = 100,000 ر.س (الراتب < 10k)
إجمالي = 126,000 + 100,000 = 226,000 ر.س
```

**الخطوة 4: القسط الشهري**
```
النسبة_الشهرية = 2.5% ÷ 12 = 0.002083
المدة = 60 شهر
القسط = 226,000 × (0.002083 × 1.002083^60) / (1.002083^60 - 1)
      ≈ 3,900 ر.س
```

**الخطوة 5: التحقق**
```
أقصى_قسط = 7,000 × 0.33 = 2,310 ر.س
القسط_الفعلي = 3,900 ر.س
⚠️ تحذير: القسط يتجاوز الحد المسموح!
```

**النتيجة:**
```
التمويل المقترح: 226,000 ر.س
القسط الشهري: 3,900 ر.س (لمدة 60 شهر)
تحذير: تحقق مع البنك للتأكد من الاستطاعة
```

## 🔟 الحالات الخاصة

### حالة 1: قريب من التقاعد جداً
```
عمر: 57 سنة (مدني)، متبقي = 3 سنوات = 36 شهر
معامل_فعلي = 18 × (36 ÷ 60) = 10.8
وليس 18 كامل!
```

### حالة 2: وصل سن التقاعد
```
عمر: 60+ سنة
متبقي = 0 أو سالب
النتيجة: لا يمكن الحصول على قرض ❌
```

### حالة 3: عسكري برتبة عالية
```
عسكري، رتبة: عميد (سن تقاعد: 60)
يشوه عمره 58 سنة
متبقي = 2 سنوات
معامل_فعلي = 18 × (24 ÷ 60) = 7.2
```
