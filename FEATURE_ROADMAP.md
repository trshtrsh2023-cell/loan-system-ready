## نقاط المساهمة

### للمطورين المهتمين:

```bash
# Fork the repo
git clone https://github.com/yourname/loan-system.git
cd loan-system

# اختر feature من Backlog
# أنشئ branch جديد
git checkout -b feature/feature-name

# اكتب الكود
# أضف الاختبارات
# اعمل commit

# Push وفتح Pull Request
git push origin feature/feature-name
```

### متطلبات PR:

- [ ] الكود يتبع معايير المشروع
- [ ] Tests مكتوبة
- [ ] Documentation محدثة
- [ ] لا توجد breaking changes
- [ ] Code review passed
